# MEG Font Family

* The MEG font family is a bitmap font family designed for use with MEG-OS.
* MEG フォントファミリーは MEG-OS で使うことを目的としてデザインされたビットマップフォントシリーズです。
* Original: https://github.com/meg-os/font-megos
